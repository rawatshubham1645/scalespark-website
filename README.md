# ScaleSpark – Social Media Marketing Agency

Website for **ScaleSpark**, a social media marketing agency. Built with HTML, CSS, and JavaScript.

## What’s in this repo

- **`index.html`** – Main page (hero, services, promise, about, contact).
- **`css/`** – Styles: `variables.css`, `styles.css`.
- **`js/`** – Scripts: `main.js`, `navigation.js`, `animations.js`, `utils.js`, `constants.js`.
- **`images/`** – All images (logo and assets).
- **`netlify.toml`** – Netlify config for deployment.

## Run locally

Open `index.html` in a browser, or use a simple local server:

```bash
# From project root (e.g. Jim Corbet folder)
python3 -m http.server 8000
# Then open http://localhost:8000
```
